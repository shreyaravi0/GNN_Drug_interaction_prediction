## âš™ï¸ Workflow

### 1. Data Preparation
- Drug pairs are read from the DrugBank dataset.
- SMILES strings are converted into molecular graphs.

### 2. Model Training
- A **Graph Neural Network** is trained using molecular graph representations.
- The trained model learns interaction patterns between drugs.

### 3. Prediction
- Users input two drug SMILES strings.
- The model predicts:
  - **Interaction label** (Interacting / Non-Interacting)
  - **Interaction probability**

### 4. Visualization
- 2D molecular structures are displayed.
- IUPAC names are retrieved and shown in the frontend.
